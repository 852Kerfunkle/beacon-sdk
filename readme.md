## Getting started

### Requirements

```
npm >= 5
NodeJS >= 8
```

Everything else gets installed automatically using `npm install`.

### Clone and Run

```
$ git clone https://github.com/airgap-it/wallet-communication-sdk.git
$ cd wallet-communication-sdk
$ npm install
$ npm test
```
